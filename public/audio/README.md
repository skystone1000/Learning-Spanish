Audio directories are prepared locally under `m1/w1` through `m6/w4`.

Lesson data references MP3 files using predictable slugs, for example:

- `/audio/m1/w1/hola.mp3`
- `/audio/m4/w2/sigue-recto.mp3`
- `/audio/m6/w4/la-conversacion.mp3`

Silent placeholder files or real recordings still need to be added before production audio playback is expected to work.
